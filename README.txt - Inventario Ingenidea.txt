README.txt - Sistema de Gestión de Inventario

¡Bienvenido al Sistema de Gestión de Inventario!

Este programa te ayudará a controlar tus productos de forma eficiente.

=======================================================
**COMPONENTES NECESARIOS PARA EL FUNCIONAMIENTO**
=======================================================

Para que el Sistema de Gestión de Inventario funcione correctamente,
necesitas OBLIGATORIAMENTE dos archivos juntos:

1.  **El archivo ejecutable (.exe):**
    * `InventarioProductos(Ingenidea).exe`
    * Este es el programa principal que usas para interactuar.

2.  **El archivo de base de datos:**
    * `basedatos2002.mdb`
    * Este archivo almacena TODA la información de tus productos, entradas y salidas.

=======================================================
**¡IMPORTANTE: DEPENDEN UNO DEL OTRO!**
=======================================================

**El ejecutable y la base de datos DEBEN estar en la MISMA CARPETA.**

* El programa (`.exe`) necesita la base de datos (`.mdb`) para poder:
    * Guardar los productos que registras.
    * Actualizar el stock cuando hay entradas o salidas.
    * Mostrarte la información en tiempo real en las casillas y tablas.

* **Si eliminas o mueves `basedatos2002.mdb`, el programa NO FUNCIONARÁ.**
    No podrá encontrar ni guardar tus datos de inventario.

* **Si eliminas o mueves `InventarioProductos(Ingenidea).exe`, no podrás abrir el programa.**

**Mantén siempre ambos archivos juntos en la misma ubicación para asegurar el correcto funcionamiento.**

=======================================================
**CÓMO USAR EL PROGRAMA**
=======================================================

1.  Asegúrate de que `InventarioProductos(Ingenidea).exe` y `basedatos2002.mdb` estén en la misma carpeta.
2.  Haz doble clic en `InventarioProductos(Ingenidea).exe` para iniciar el programa.
3.  Sigue las opciones del menú para gestionar tu inventario.
